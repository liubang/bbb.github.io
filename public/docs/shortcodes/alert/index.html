Alert Shortcode

2020-10-22 1 分钟阅读 ShortcodeDocsAlert
This theme provides several kinds of layouts, such as posts and docs. Our documentation uses the docs layout. If you're looking for an example that using posts layout, please take a look at the page Markdown Syntax.

This article shows how to use the alert shortcode.

1{{< alert "Message" [type] >}}

The parameter type is optional. Default to info.

Info

1{{< alert "Info" >}}
Info

Success

1{{< alert "Success" success >}}
Success

Warning

1{{< alert "Warning" warning >}}
Warning

Danger

1{{< alert "Danger" danger >}}
Danger
